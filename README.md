# home_budget
A tool created in Excel that helps you with analysis of your household budget.
In the spreadsheet named 'Charts' you can find one pivot table, one pivot chart, and to normal charts.
Cells A11 and A39 allow us to manipulate the data displayed on normal charts.
The spreadsheet ‘Summary’ informs us about the spendings and enrings during each month. 
It updates itself automatically based on values inserted in sheets ‘Income’ and ‘Expanses’.
